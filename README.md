## Description

This project focuses on layout, content structure, and visual hierarchy using HTML and CSS. For Portfolio 1, usability was imporved by adding JavaScript functionality without redesigning the site.

Improvements made:
- Added exhibiton category filter buttons
- Used data attributes to tag exhibition cards
- Implemented DOM-based filtering using JavaSCript event listeners.

## Built with:

- HTML
- CSS
- Javascript (Vanilla)
